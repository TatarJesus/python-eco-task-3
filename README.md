2. Инструкция по запуску

* Создание и активация виртуального окружения Python:

```shell
python -m venv venv
venv\Scripts\activate
```

* Установка пакетов:

```shell
pip install -r requirements.txt
```

* Запуск сервера:

```shell
uvicorn main:app --host 0.0.0.0 --port 8000
```
